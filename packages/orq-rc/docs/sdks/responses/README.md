## create

Creates a model response for the given input. Returns a response object or a stream of server-sent events.

### Example Usage

<!-- UsageSnippet language="python" operationID="create-router-response" method="post" path="/v3/router/responses" -->
```python
from orq_ai_sdk import Orq
import os


with Orq(
    api_key=os.getenv("ORQ_API_KEY", ""),
) as orq:

    res = orq.responses.create()

    with res as event_stream:
        for event in event_stream:
            # handle event
            print(event, flush=True)

```

## get

Retrieves a previously created response by its ID.

### Example Usage

<!-- UsageSnippet language="python" operationID="retrieve-response" method="get" path="/v3/router/responses/{response_id}" -->
```python
from orq_ai_sdk import Orq
import os


with Orq(
    api_key=os.getenv("ORQ_API_KEY", ""),
) as orq:

    res = orq.responses.get(response_id="<id>")

    # Handle response
    print(res)

```
