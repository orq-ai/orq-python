# Status

Comma-separated list of task statuses to filter by. Available values: inactive, approval_required, in_progress, errored


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `SUBMITTED`      | submitted        |
| `WORKING`        | working          |
| `INPUT_REQUIRED` | input-required   |
| `AUTH_REQUIRED`  | auth-required    |
| `COMPLETED`      | completed        |
| `FAILED`         | failed           |
| `CANCELED`       | canceled         |
| `REJECTED`       | rejected         |
