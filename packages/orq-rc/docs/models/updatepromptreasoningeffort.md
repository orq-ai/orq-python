# UpdatePromptReasoningEffort

Constrains effort on reasoning for reasoning models. Reducing reasoning effort can result in faster responses and fewer tokens used on reasoning in a response.


## Values

| Name      | Value     |
| --------- | --------- |
| `DISABLE` | disable   |
| `LOW`     | low       |
| `MEDIUM`  | medium    |
| `HIGH`    | high      |
