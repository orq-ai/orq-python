# OutputFormat

The format in which the generated images are returned. This parameter is only supported for `openai/gpt-image-1`.

## Example Usage

```python
from orq_ai_sdk.models import OutputFormat
value: OutputFormat = "png"
```


## Values

| Name   | Value  |
| ------ | ------ |
| `PNG`  | png    |
| `JPEG` | jpeg   |
| `WEBP` | webp   |
