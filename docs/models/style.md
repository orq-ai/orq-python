# Style

The style of the generated images. This parameter is only supported for `openai/dall-e-3`. Must be one of `vivid` or `natural`.

## Example Usage

```python
from orq_ai_sdk.models import Style
value: Style = "vivid"
```


## Values

| Name      | Value     |
| --------- | --------- |
| `VIVID`   | vivid     |
| `NATURAL` | natural   |
