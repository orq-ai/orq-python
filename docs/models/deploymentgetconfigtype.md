# DeploymentGetConfigType

The type of the model. Current `chat`,`completion` and `image` are supported


## Values

| Name          | Value         |
| ------------- | ------------- |
| `CHAT`        | chat          |
| `COMPLETION`  | completion    |
| `EMBEDDING`   | embedding     |
| `IMAGE`       | image         |
| `TTS`         | tts           |
| `STT`         | stt           |
| `RERANK`      | rerank        |
| `MODERATIONS` | moderations   |
| `VISION`      | vision        |
