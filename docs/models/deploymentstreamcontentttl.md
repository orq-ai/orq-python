# DeploymentStreamContentTTL

The time-to-live for the cache control breakpoint. This may be one of the following values:

- `5m`: 5 minutes
- `1h`: 1 hour

Defaults to `5m`. Only supported by `Anthropic` Claude models.


## Values

| Name    | Value   |
| ------- | ------- |
| `FIVEM` | 5m      |
| `ONEH`  | 1h      |
