# CreateTranscriptionResponseFormat

The format of the transcript output, in one of these options: json, text, srt, verbose_json, or vtt.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `JSON`         | json           |
| `TEXT`         | text           |
| `SRT`          | srt            |
| `VERBOSE_JSON` | verbose_json   |
| `VTT`          | vtt            |
