# Quality

The quality of the image that will be generated. Auto will automatically select the best quality for the given model.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `AUTO`     | auto       |
| `HIGH`     | high       |
| `MEDIUM`   | medium     |
| `LOW`      | low        |
| `HD`       | hd         |
| `STANDARD` | standard   |
