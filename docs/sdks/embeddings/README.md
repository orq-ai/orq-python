# Embeddings
(*router.embeddings*)

## Overview

## create

Creates an embedding vector representing the input text.

```python

import os

    api_key=os.getenv("ORQ_API_KEY", ""),
)

res = s.router.embeddings.create(request={
    "input": "<value>",
    "model": "LeBaron",
})

if res is not None:
    # handle response
    pass

```
