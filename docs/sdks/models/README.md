## list

Lists the currently available models, and provides basic information about each one such as the owner and availability.

### Example Usage

```python
from orq_ai_sdk import Orq
import os


with Orq(
    api_key=os.getenv("ORQ_API_KEY", ""),
) as orq:

    res = orq.models.list()

    assert res is not None

    # Handle response
    print(res)

```
